# Nyster

## Getting Started
Nyster allows you to integrate with GitHub and GitLab through a variety of commands. Before you begin using the commands, set up your tokens:

- **GitHub Setup**: Run the `/github-setup` command and provide your GitHub token. Ensure the token has read-only access.
- **GitLab Setup**: Run the `/gitlab-setup` command and provide your GitLab token. Ensure the token has read-only access.

## Available Commands

### GitHub Commands
1. **`/github-setup`**: Set up your GitHub token.
2. **`/github-setup-remove`**: Remove your GitHub token.
3. **`/github-status`**: Check the status of your GitHub integration.
4. **`/github-commits`**: Fetch the latest commits from a repository.
5. **`/github-repo-info`**: Display detailed information about a specific GitHub repository.
6. **`/github-track-commits`**: Track commits of a repository and receive notifications.
7. **`/github-untrack-commits`**: Stop tracking commits of a repository.
8. **`/github-track-releases`**: Track new releases of a repository.
9. **`/github-untrack-releases`**: Stop tracking releases of a repository.

### GitLab Commands
1. **`/gitlab-setup`**: Set up your GitLab token.
2. **`/gitlab-setup-remove`**: Remove your GitLab token.
3. **`/gitlab-status`**: Check the status of your GitLab integration.
4. **`/gitlab-repo-info`**: Display detailed information about a specific GitLab repository.
